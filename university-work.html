DirectX 11 Physics Engine

This Project is a physics engine that I made in DirectX11, that implements a mass aggregate physics system and other physics elements. The engine has particle models that control objects and make them physics based, this allows objects to have acceleration, velocity, thrust, gravity, net forces and buoyancy. The project also has rigidbodies that utilise inertia and torque in combination with quaternion calculations to create, apply and simulate physics rotations. The project also has support for particle systems, that are also controlled via physics and similarly to the objects can have elements like acceleration and velocity. The project also includes object collisions, this includes AABB, Sphere, Point, Bounding Box etc. And finally, it has all the capabilities to create large amounts of objects with proper management and disposal including a rendering system.

SDL Mario

The game has 4 different levels, and a level selector. It can be played singleplayer or multiplayer through local play. The game has a score saving system, two way blocks, lucky blocks that speed the player up adn screen warping. It also has 3 types of enemies that gradually progress through the map and can be killed by jumping on them when they're knocked up. Hitting pow blocks will knock enemies up and create a screen shake and visual effect. Levels are created via a TXT file and are loaded dynamically.
