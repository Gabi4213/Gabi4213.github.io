DirectX11 Physics Simulation/Engine

Overview: A custom-built physics and rendering engine developed in C++ using DirectX 11. This project was created over the course of a few weeks as part of a university assignment, with the goal of learning low-level systems and replicating features found in modern engines like Unity. The engine includes a real-time physics simulation system with support for rigidbodies, forces, collisions, and rotational dynamics, as well as a flexible rendering pipeline capable of drawing both 2D and 3D objects. This project not only helped me gain a deeper understanding of graphics and physics programming, but also gave me the chance to prove to myself and others that I can take on large scale technical challenges and bring them to life from scratch.

Key Features:

  • Rigidbodies with dynamic and static modes
  • Linear Motion: velocity, acceleration, net force, and drag
  • Force Application: supports continuous and instantaneous forces
  • Buoyancy Simulation: basic floating object behavior in fluids
  • Angular Motion: angular velocity, angular acceleration, and torque
  • Inertia & Moment of Inertia for rotational dynamics
  • Collision Detection: AABB, sphere, and mixed-type intersections
  • Collision Resolution: impulse-based response with mass handling
  • Particle Systems with full physics support
  • Torque-based Rotation from force offsets
  • Semi-implicit Euler Physics Integration
  • Scalable architecture ready for joints, springs, and constraints

Overview:A physical smart chessboard system combining 3D printing, Arduino, and Raspberry Pi to create an interactive chess experience. The board uses LED strips to illuminate the 8x8 grid, highlighting legal moves, last moves, and AI responses in real time. An Arduino Nano controls the LED matrix and button inputs for each square, while a Raspberry Pi runs the main chess engine and game logic, allowing players to challenge a local AI opponent. The system also includes state detection and serial communication between the microcontroller and the Pi to synchronize game state. This project taught me valuable lessons in embedded systems, hardware-software integration, and designing interactive physical interfaces.

Key Features:

  • Individually addressable LEDs light up the 8x8 board to show legal moves, selected pieces, and AI responses
  • Integrated custom chess engine running on a Raspberry Pi
  • Manages LED animations and button input for each square with real-time responsiveness
  • Handles move calculation, game state management, and communicates with the Arduino via serial.
  • LEDs dynamically indicate player and AI moves
