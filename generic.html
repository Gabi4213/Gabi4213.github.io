Citrus Intolerant

Citrus Intolerant is a mobile game that me and my friend Avery Cross developed and published. We worked on this game for about 2 weeks and published it the following one. It is a hyper casual game where the player needs to try and get the highest score by propelling their player forwards. The player at the same time needs to watch their milk meter, because if this reaches 0 the game will be lost. The game features a full fledged shop where the user can customise their character. We also implemented achievments, rewards and a real time leaderboard system. Advertisements have also been integrated into the game. This takes form in a "watch an ad to continue" manner, where if the player dies he/she has a choice of continuing the game by watching an ad. I'm proud of this game as it is the first game that I have commercially published (not including itch.io publications). Though I would say the game could do with a rework to make it even better.This game is currently avalibale on PC which can be downlaoded at my Itch.io page. But it is also avalibale on the Google play store for android.

Nightmares

Nightmares is a game created for the 3rd Blackthornprod game jam. It was created by Zyger and Syviery within the game jam period (around 6 days). This is a puzzle platforming game, that takes place in the players nightmare. where he has to learn a valuable lesson of less is more. The main mechanic of the game is the ability of comparing object to one another. For example making objects smaller, larger or equal to each other depending on its original size. The game is momentum based therefore the player needs to be careful not to move to fast in trickier situations.
